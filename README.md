# 🚀 Projetos em destaque 

## **Gerenciador de Eventos**

API backend desenvolvida em Java com foco na gestão de eventos, cupons e endereços.

- 🔧 Java + Spring Boot
- 🗄️ PostgreSQL + Flyway (migrations)
- ☁️ AWS (EC2, S3 e RDS)
- 🧪 Testes de regras de negócio com JUnit
- 🖼️ Upload e manipulação de imagens (PNG/JPG)
- 📦 Arquitetura MSC (Model, Service, Controller)
- 🔄 Uso de DTOs e mapeamento manual
- 📄 Paginação com Pageable

📌 Funcionalidades:

CRUD completo para eventos
CRUD de cupons com regras específicas
Gerenciamento de endereços
Controllers separados por domínio (eventos e cupons)

🔗 Link do respositório: https://github.com/Joao-Pedro-SA/EventManager_API.git

## **Gerenciador de atividades**
API REST para gerenciamento de tarefas, com foco em boas práticas de desenvolvimento e testes.

- 🔧 Java + Spring Boot
- 🐳 Docker (imagem publicada no Docker Hub)
- 🧪 Testes com JUnit (controller, service e banco de dados)
- 📘 Swagger/OpenAPI para exploração dos endpoints
- 📦 Arquitetura MSC (Model, Service, Controller)
- 🔄 DTOs e mapeamento manual
- 📄 Paginação com Pageable

📌 Funcionalidades:

CRUD completo de atividades
Endpoints testados e organizados
Interface Swagger disponível para testes da API

🔗 link do repositório: https://github.com/Joao-Pedro-SA/task-management-api.git
